## NestJS Study

- NestJS 스터디용 Repository

## Description

- Typescript
- Postgresql by Docker
- TypeOrm
- JWT
- Class Validator, Transformer

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
