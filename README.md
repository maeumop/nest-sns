## NestJS Study

- NestJS 스터디용 Repository

## Description

- Typescript
- Postgresql by Docker
- TypeOrm
- JWT
- class Validator, transformer

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
